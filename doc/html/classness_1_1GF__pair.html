Detailed Description

Class GF_pair represents Green's functions in both time or frequency domain.

Purpose

This class contains two GF data structures for representing complex matrix valued Green's functions on a time and frequency grid (retarded and lesser component), and provides a FFT function given by the helper class fft_solver for calculating Fourier integral transforms.
