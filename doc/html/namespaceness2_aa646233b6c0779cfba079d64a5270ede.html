Calculate the bubble diagram of first type: \( C_{c_1,c_2} (t)= i A_{a_1,a_2} (t) B_{b_2,b_1} (-t) \)

Purpose

Evaluate Green's function C given by the bubble diagram equation \( C_{c_1,c_2} (t)= i A_{a_1,a_2} (t) B_{b_2,b_1} (-t) \). The evaluation is done for all time points and for the lesser and retarded component.
