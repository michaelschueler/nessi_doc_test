Solve the Dyson equation for the steady-state Green's function.

Purpose

Computes the time-domain solution \( G^{R|<}(t) \) of the Dyson equation for a steady-state system, using a frequency-domain method with optional bath regularization. The method transforms \( \Sigma^{R|<}(t) \) to frequency space, solves Dyson equations there, and back-transforms to obtain \( G^{R|<}(t) \).

Parameters
[out]GOutput Green's function \( G^{R|<}(t) \) of type herm_matrix_ness, modified in-place.
[in]muChemical potential \( \mu \).
[in]epsilonComplex-valued single-particle Hamiltonian matrix \( \epsilon \) (dimension must match G).
[in]SigmaInput self-energy \( \Sigma^{R|<}(t) \) of type herm_matrix_ness, defined on the same grid as G.
[in]hTime step size \( h \) used for time grids.
[in]methodIntegral transformation method (default = FFT_TRAPEZ). Currently only FFT_TRAPEZ is implemented.
[in]bath_typeType of bath regularization added to the self-energy. If bath_type != REG_NONE, a bath self-energy \( \Sigma_{\rm reg} \) is added to \( \Sigma \):
  • REG_CONST: Constant density (Lorentzian broadening)
  • REG_GAUSS: Gaussian cutoff in frequency space.
  • REG_OHMIC: Ohmic spectral function with cutoff.
[in]bath_etaBroadening parameter \( \eta \) for the bath (used for all regularization types except BATH_NONE).
[in]bath_betaInverse temperature \( \beta \) of the bath (used in BATH_CONST, BATH_GAUSS, BATH_OHMIC).
[in]bath_muChemical potential \( \mu_{\rm bath} \) of the bath (used in BATH_CONST, BATH_GAUSS).

References BATH_NONE, and FFT_TRAPEZ.
