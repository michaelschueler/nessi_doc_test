double ness2::distance_norm2 ( fft_array & A,
fft_array & B,
fft_domain type )

Compute Euclidean distance between two fft_array objects in time or frequency domain

Purpose

Computes the L2 norm (Euclidean distance) \( \|A - B\|_2 = \sqrt{ \sum_i |A_i - B_i|^2 } \) between two fft_array objects in either time or frequency domain.

Parameters
&A
First fft_array object
&B
Second fft_array object
type
fft_domain::time or fft_domain::freq – determines which internal data to compare
Returns

Euclidean norm of the difference
Exceptions
std::invalid_argument
If array sizes or shapes do not match
