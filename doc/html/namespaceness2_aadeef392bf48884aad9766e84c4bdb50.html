Equilibrium steady-state Green's function systems

Constructs a retarded and lesser Green's function in the time domain for a system in thermal equilibrium, using a specified density of states and numerical Fourier integration method.

Template Parameters
DOSA density of states class providing:
  • double lo_ — lower frequency bound
  • double hi_ — upper frequency bound
  • double operator()(double omega) — density of states \( A(\omega) \)
Parameters
bose_fermi= BOSON (+1) or FERMION (-1) sign
GOutput: Green's function object (herm_matrix_ness) to be filled in time domain
dosDensity of states object used to define the spectral function
betaInverse temperature \( \beta \)
hTime step size
muChemical potential \( \mu \)
methodIntegration method to use for the Fourier transform. One of:
  • FFT_TRAPEZ: Simple piecewise-constant quadrature (trapezoidal rule)
  • FFT_ADAPTIVE: Adaptive quadrature as in libcntr (uses nn and limit)
nnNumber of sub-sample points per interval (used only if method == FFT_ADAPTIVE)
limitMaximum number of intervals in adaptive Fourier transform (used only if method == FFT_ADAPTIVE)

Computes the time-domain Green's functions:

\[G^R(t) = -i \int_{\mathrm{lo}}^{\mathrm{hi}} d\omega\, A(\omega) e^{-i \omega t} \]

\[G^<(t) = - \xi i \int_{\mathrm{lo}}^{\mathrm{hi}} d\omega\, A(\omega) f_F(\omega - \mu) e^{-i \omega t}, \quad f_F(x) = \frac{1}{e^{\beta x} - \xi} \]

where $\xi$ = BOSE or FERMI, and \( A(\omega)\) is computed from the density of states.

The integration bounds are taken from dos.lo_ to dos.hi_.

For FFT_ADAPTIVE, a frequency-adaptive scheme is used, refining the interval until convergence. For FFT_TRAPEZ the FFT grid defined by Nft_ and h is used, nn and limit are ignored
