int find_param ( std::string file_name,
std::string param,
int & value )

Function for reading out an integer type parameter of a given input file.

Purpose

Parse an input file of given name (e.g. given by argv[1]) for a parameter (given in the input file in the format __parametername==parametervalue) and writing the value into an integer.

Parameters
file_name

[std::string] Name of the input file.

param[std::string] Name of the parameter in the input file, in the format __parametername.
&value[int] Integer to store the parameter value in.
