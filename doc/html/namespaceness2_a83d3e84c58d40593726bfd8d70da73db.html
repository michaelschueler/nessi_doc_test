template<typename T>
void ness2::cntr2ness ( herm_matrix_ness & Gness,
cntr::herm_matrix< T > & Gcntr,
int tstp = -1 )

Convert CNTR Green's function cntr::herm_matrix to NESS ness2::herm_matrix_ness by sampling a given time slice.

Read \( G_{\rm ness}^{R|<}(t) = G{\rm cntr}^{R|<}(tstp, tstp - t) \) from CNTR object at time slice tstp, and write to NESS object.

Values at negative times \( t < 0 \) are set by Hermitian symmetry: \( G^<(-t) = -[G^<(t)]^\dagger \)

If tstp is smaller than the maximum time in Gness, the tail is left zero.
