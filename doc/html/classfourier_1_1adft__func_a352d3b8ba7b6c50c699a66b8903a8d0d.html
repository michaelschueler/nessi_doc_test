Fourier samples the given function to determine the individual intervals.

Purpose

Computes the Fourier integral for fixed frequency \(\omega\). The interval \([a,b]\) is iteratively split until the error of the Fourier integral over each subinterval is smaller or equal than the smallest error.

Parameters
w

frequency \(\omega\) in the Fourier integral

alower interval bound
bupper interval bound
fz[template class function] scalar complex function
nnumber of points in each interval
limitmax number of intervals
