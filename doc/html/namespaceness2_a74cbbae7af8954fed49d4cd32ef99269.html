template<typename T>
void ness2::ness2cntr ( cntr::herm_matrix< T > & Gcntr,
const herm_matrix_ness & Gness )

Convert a steady-state Green's function into a CNTR Green's function, assuming tranlstional invariance in time.

For each pair \( (t, t') \), if the relative time \( t - t' \) exists in the NESS domain, use \( G_{cntr}(t,t') = G_{ness}(t - t') \). Else, set to zero.
