herm_matrix_ness ness2::downsample ( herm_matrix_ness & in,
int factor )

Downsampling of a herm_matrix_ness to a coarser time grid, taking only every factor entry of the original object.

The new length is \( N_{ft1} = \frac{N_{ft}}{\tt factor} \). In frequency space, only values \( 0, \ldots, \frac{N_{ft1}}{2} - 1, N_{ft} - \frac{N_{ft1}}{2}, \ldots, N_{ft} - 1 \) of the input are written to the output object. Returns a downsampled herm_matrix_ness object. It is required that \( N_{ft} \) is an integer multiple of factor, and \( N_{ft1} \) (the new length) is still an integer multiple of 2.

Parameters
inInput steady-state Green's function (herm_matrix_ness).
factorCoarse-graining factor, larger than one.
