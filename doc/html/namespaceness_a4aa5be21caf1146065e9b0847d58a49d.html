Equilibrium steady state propagator for the given density of states

Purpose

Calculate the steady state equilibrium GF_pair \( G \) (in time domain) for the given density of states (own type dos_function) via \(G(t-t') = -i \int d\omega A(\omega+\mu) \exp(i\omega (t'-t)) [ \Theta(t-t') - \Theta(t'-t)\exp(-\beta*\omega)]\), for all times and the retarded and lesser component. The calculation uses an adaptive Fourier transform algorithm.

Parameters
GThe output Greens function pair GF_pair set to the equilibrium free propagator (time component)
dosdensity of states
betainverse temperature
htimestep
muchemical potential
limitmax number of intervals in Fourier transform (default: 100)
nnnumber of points in each interval of the Fourier transform (default: 20)
